# Всплывающая подсказка

## Возможности

- Всплывающие подсказки при клике
- Разные позиции tooltip:
  - top (по умолчанию)
  - left
  - right
  - bottom
- Закрытие:
  - по повторному клику
  - по клику вне элемента
- Динамическое создание DOM-элементов


## Как работает

1. Клик по ссылке `.has-tooltip`
2. Берётся текст из `title`
3. Создаётся `<div class="tooltip">`
4. Вычисляется позиция через `getBoundingClientRect()`
5. Tooltip позиционируется через `top/left`
6. Старый tooltip удаляется при новом клике


## Технологии

- HTML5
- CSS3
- JavaScript (DOM, events, dataset, positioning)
