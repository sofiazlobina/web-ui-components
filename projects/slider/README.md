# Слайдер изображения

Слайдер изображений, написанный с помощью технологий **HTML, CSS и JavaScript** без использования библиотек и фреймворков.


## Возможности

- Переключение слайдов кнопками (left-right)
- Навигация по точкам (dots)
- Зацикленная прокрутка (infinite loop)
- Подсветка активного слайда и точки


## Как работает

- Все слайды скрыты по умолчанию (`display: none`)
- Активный слайд получает класс `slider__item_active`
- JS управляет индексом текущего слайда (`currentIndex`)
- При кликах обновляется активный класс у:
    - слайдов
    - точек навигации


## Технологии

- HTML5
- CSS3 (Flexbox, positioning)
- JavaScript (DOM, events, arrays)


## Запуск проекта

Открыть файл
```
index.html
```


## Логика переключения

- `prev` - `currentIndex - 1`
- `next` - `currentIndex + 1`
- выход за границы - переход в начало/конец
- точки - прямой переход к слайду
