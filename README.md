# Коллекция интерактивных UI‑компонентов

Демонстрация реализации различных интерактивных элементов на HTML, CSS и JavaScript.

## Доступные проекты

* **Всплывающая подсказка** — подсказка на CSS/JS.
* **Курс валют** — загрузка данных с API и анимация загрузки.
* **Слайдер** — интерактивный слайдер с возможностью переключения изображений.
